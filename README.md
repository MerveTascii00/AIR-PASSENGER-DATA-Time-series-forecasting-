# Time Series Forecasting Project

This project was developed for a time series forecasting assignment using Python.

## Objective

The aim of the project is to analyze and forecast monthly passenger traffic data with different time series methods.

## Dataset

The dataset contains monthly air passenger traffic data.
For this project, JFK Airport observations were selected.

Source: Kaggle - NYS Air Passenger Traffic, Port Authority of NY & NJ

## Completed Sections

So far, the following parts of the project have been completed:

1. Data understanding and time series visualization
2. 2-period and 3-period moving average
3. 2-period and 3-period centered moving average

## Project Structure

- `notebooks/01_data_understanding.ipynb`
- `notebooks/02_moving_average.ipynb`
- `notebooks/03_centered_moving_average.ipynb`
